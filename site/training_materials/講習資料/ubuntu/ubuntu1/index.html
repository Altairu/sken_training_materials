    UbuntuгҒЁгҒҜҒEҒE/strong>

    • гӮӘгғјгғ—гғігӮҪгғјгӮ№гҒ®LinuxгғҒEӮЈгӮ№гғҲгғӘгғ“гғҘгғјгӮ·гғ§гғі
    • гғҒEӮ№гӮҜгғҲгғғгғ—гҖҒгӮөгғјгғҗгҖҒгӮҜгғ©гӮҰгғүгҒӘгҒ©е№ҒEәҒEҒ„з”ЁйҖ”гҒ«еҜҫеҝҒE/li>
    • CanonicalзӨҫгҒҢй–ӢзҷәгҖҒгӮөгғқгҒEгғҒE/li>

    еҹәжң¬гӮігғһгғігғҒE/strong>

    • ls: гғҒEӮЈгғ¬гӮҜгғҲгғӘгҒ®еҶҒE®№гӮ’иЎЁзӨә

    ls
    - cd: гғҒEӮЈгғ¬гӮҜгғҲгғӘгӮ’еӨүжӣҙ 
    cd /path/to/directory
    - cp: гғ•гӮЎгӮӨгғ«гӮ’гӮігғ”гҒEгҒ—гҒҰиІјгӮҠд»ҳгҒ‘ 
    cp /path/to/sourcefile /path/to/destination/
    - дҫҒE/strong>: cp /home/user/file.txt /home/user/backup/ - жҢҒE®ҡгҒ—гҒҒEcode>sourcefileгӮҒEcode>destinationгғҒEӮЈгғ¬гӮҜгғҲгғӘгҒ«гӮігғ”гҒEгҒ—гҒҫгҒҷгҖҒE/p>
    • mv: гғ•гӮЎгӮӨгғ«гӮ’з§»еӢ•гҒҫгҒҹгҒEеҗҚеүҚгӮ’еӨүжӣҙ 
      mv oldname newname
    • дҫҒE/strong>: mv /home/user/file.txt /home/user/backup/file.txt

    • гғ•гӮЎгӮӨгғ«гӮ’жҢҮе®ҡгҒ—гҒҹе ҙжүҖгҒ«з§»еӢ•гҒҷгӮӢгҒӢгҖҒеҗҚеүҚгӮ’еӨүжӣҙгҒ—гҒҫгҒҷгҖҒE/p>

    • rm: гғ•гӮЎгӮӨгғ«гӮ’еүҠйҷӨ 

      rm filename

    • дҫҒE/strong>: rm /home/user/file.txt

    • жҢҒE®ҡгҒ—гҒҹгғ•гӮЎгӮӨгғ«гӮ’еүҠйҷӨгҒ—гҒҫгҒҷгҖҒE/p>

    • mkdir: гғҒEӮЈгғ¬гӮҜгғҲгғӘгӮ’дҪңжҒE 

      mkdir newdirectory

    • дҫҒE/strong>: mkdir /home/user/newfolder

    • ж–°гҒ—гҒ„гғҒEӮЈгғ¬гӮҜгғҲгғӘгӮ’дҪңжҒEгҒ—гҒҫгҒҷгҖҒE/p>

    • rmdir: гғҒEӮЈгғ¬гӮҜгғҲгғӘгӮ’еүҠйҷӨ 

      rmdir directoryname

    • дҫҒE/strong>: rmdir /home/user/oldfolder

    • жҢҒE®ҡгҒ—гҒҹгғҮгӮЈгғ¬гӮҜгғҲгғӘгӮ’еүҠйҷӨгҒ—гҒҫгҒҷпјҲз©әгҒ§гҒӮгӮӢеҝҒEҰҒгҒҢгҒӮгӮҠгҒҫгҒҷпјүгҖҒE/p>

    • rm -r: гғҒEӮЈгғ¬гӮҜгғҲгғӘгҒЁгҒқгҒEеҶҒE®№гӮ’еҒEеё°зҡҒEҒ«еүҠйҷӨ 

      rm -r directoryname

    • дҫҒE/strong>: rm -r /home/user/oldfolder

    • жҢҒE®ҡгҒ—гҒҹгғҮгӮЈгғ¬гӮҜгғҲгғӘгҒЁгҒқгҒEдёӯгҒ®е…ЁгҒҰгҒ®гғ•гӮЎгӮӨгғ«гҒҠгӮҲгҒігӮөгғ–гғҮгӮЈгғ¬гӮҜгғҲгғӘгӮ’еүҠйҷӨгҒ—гҒҫгҒҷгҖҒE/p>

    • touch: з©әгҒ®гғ•гӮЎгӮӨгғ«гӮ’дҪңжҒEгҖҒгҒҫгҒҹгҒEж—ўеӯҳгҒEгғ•гӮЎгӮӨгғ«гҒ®гӮҝгӮӨгғ гӮ№гӮҝгғігғ—гӮ’жӣҙж–° 

      touch filename

    • дҫҒE/strong>: touch /home/user/newfile.txt

    • жҢҒE®ҡгҒ—гҒҹеҗҚеүҚгҒEз©әгҒ®гғ•гӮЎгӮӨгғ«гӮ’дҪңжҒEгҒ—гҒҫгҒҷгҖҒE/p>

    • gedit: гғҒEӮӯгӮ№гғҲгӮЁгғҒEӮЈгӮҝгӮ’дҪҝз”ЁгҒ—гҒҰгғ•гӮЎгӮӨгғ«гӮ’з·ЁйӣҒE

      gedit filename

    • дҫҒE/strong>: gedit /home/user/newfile.txt

    • GNOMEгғҒEӮ№гӮҜгғҲгғғгғ—з’°еўҒEҒEгғҒEӮӯгӮ№гғҲгӮЁгғҒEӮЈгӮҝгҒ§жҢҒE®ҡгҒ—гҒҹгғ•гӮЎгӮӨгғ«гӮ’й–ӢгҒҚгҒҫгҒҷгҖҒE/p>

    • USBзўәиӘҒE

      ls -l /dev/serial/by-id/
