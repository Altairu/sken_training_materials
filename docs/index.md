# ようこそ、Sken 技術資料へ


この Wiki は、Skenで実施する各種講習資料や課題をひとまとめにし、効率的に学習できる環境を提供するために作成されました。

- **ナビゲーションバー:**  
  サイト上部のメニューから各カテゴリの講習資料や課題にワンクリックでアクセス可能です。
